# AGPI — Application de Gestion de Projets Intelligente

## Présentation du projet

AGPI est une application web conçue pour centraliser et simplifier la gestion de projets en équipe. Elle intègre des fonctionnalités intelligentes pour améliorer la coordination, réduire les tâches manuelles et optimiser le suivi des projets.

## Problématique

Les équipes projet perdent un temps considérable à jongler entre plusieurs outils (emails, tableurs, réunions) sans vue unifiée sur l'avancement. Cette fragmentation entraîne des erreurs, des retards et un manque de visibilité globale.

## Objectifs

- Centraliser la gestion des tâches, des délais et des équipes en un seul endroit
- Réduire de 30% le temps consacré à la coordination manuelle
- Offrir une vue claire et en temps réel sur l'état d'avancement des projets
- Faciliter la collaboration entre les membres d'une équipe

## Fonctionnalités prévues

- Tableau de bord avec vue Gantt et Kanban
- Score de risque intelligent par projet
- Gestion des rôles et des membres
- Notifications et alertes automatiques

## Stack technique

- **Frontend** : HTML, CSS, JavaScript
- **Versioning** : Git / GitHub
